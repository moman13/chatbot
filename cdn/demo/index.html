Widget Settings

üìñ Configuration Options

Basic Options

  • botName - Display name
  • welcomeMessage - First message
  • placeholder - Input placeholder
  • apiEndpoint - Chat API URL

Styling Options

  • primaryColor - Main accent color
  • secondaryColor - Secondary accent
  • darkMode - Enable dark theme
  • borderRadius - Corner radius

Layout Options

  • position - Widget placement
  • width - Widget width (px)
  • height - Widget height (px)
  • zIndex - Stacking order

Behavior Options

  • enableTyping - Typing indicators
  • typingDelay - Response delay
  • maxMessages - Message limit
  • enableSound - Sound effects

Advanced Configuration

<script> const widget = AqlBot.init({ botName: 'Customer Support', welcomeMessage: 'Hello! How can we help you today?', primaryColor: '#3b82f6', secondaryColor: '#1e40af', position: 'bottom-left', darkMode: false, width: 400, height: 600, enableTyping: true, typingDelay: 800, apiEndpoint: 'https://api.your-domain.com/chat', showBranding: false }); // Programmatic control widget.open(); // Open chat widget.close(); // Close chat widget.send('Hello from code!'); // Send message </script>
